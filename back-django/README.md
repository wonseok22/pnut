## 가상환경 설정

---

- **`$ python -m venv venv`**
    - 가상환경 생성
- **`$ source ./venv/Scripts/activate`**
    - 가상환경 활성화

## 패키지 설치 및 관리

---

- **`$ pip list`**
    - 현재 패키지 목록 출력
- **`$ pip install -r requirements.txt`**
    - requirements.txt 파일 바탕으로 패키지 설치
- **`$ pip freeze > requirements.txt`**
    - 새로운 패키지 설치 시 requirements.txt에 업데이트
